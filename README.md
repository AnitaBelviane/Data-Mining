# Data-Mining
Ce dépôt contient l’ensemble des travaux pratiques de Fouille de données (Data Mining). Les exercices sont réalisés principalement sur le jeu de données Wine (UCI Repository). Thèmes abordés :  Analyse exploratoire (EDA), Discrétisation des variables, Extraction de règles d’association (Min-Apriori, FP-Growth, SPADE) ,Clustering avec DBSCAN
